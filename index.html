# HERO Pages Mapa lógico atual das páginas/telas do HERO. A execução permanece no `index.html` para não alterar o comportamento do produto nesta etapa. Telas identificadas: - Login - Painel do treinador - Cadastro/edição de aluna - Prescrição de treino - Área da aluna - Comunidade HERO - Biblioteca de exercícios - Chat/IA HERO Quando a refatoração funcional for autorizada, cada tela poderá ser extraída para arquivo próprio mantendo as mesmas funções e contratos de dados.
